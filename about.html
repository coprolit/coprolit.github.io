About

Welcome to the Command Post - a collection of tools and content to spice up your Bolt Action game.

The Command Post is a hobby project made by Bolt Action fans.

About to start up a Bolt Action game?

Use the Generator to roll a random Scenario from your choice of official and unofficial Scenarios.

Include our unofficial missions to add extra tactical challenge and decision making to your game. Pick a mission manually, or use the Generator to get random secret missions assigned to each player.

Work in progress

We have plans to expand the service to include battlefield conditions, nationality based missions and more.

Please note that as of now this is a work in progress - therefore we kindly ask that you submit any and all thoughts, ideas and any issues that you encounter while using the tool.

Thanks a lot for giving it a try.
